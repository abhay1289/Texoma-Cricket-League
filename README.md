# Texoma Cricket League

Premier youth cricket tournament platform for Texas & Oklahoma.

## Features
- Youth cricket tournaments
- Academy partnerships
- Professional coaching
- State-of-the-art facilities

## Tech Stack
- Next.js 14
- Tailwind CSS
- Framer Motion

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
